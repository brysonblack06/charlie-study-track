# StudyTrack

> Plan your study time, record your progress, and review your work.

StudyTrack is a front-end web application that allows students to
record and review study sessions. Users can enter a subject, date,
duration, session type, and notes. The application calculates total
study time and saves entries in the browser using localStorage.

## User Story

As a college student, I want to record my study sessions so that I can
see how much time I spend studying and keep track of the subjects I
have worked on.
